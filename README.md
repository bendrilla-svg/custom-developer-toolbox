# Custom Developer Toolbox

A centralized, modular automation package engineered to simplify workspace administration, file cleanup operations, and automated text extraction workflows.

## Key Utilities
*   **File Cleaner:** Automates system housekeeping by sweeping out temporary storage areas and preserving active project resources.
*   **Text Parser:** Instantly extracts target milestones or lines from large plain-text log files.

## Easy Setup Guide (For Non-Technical Users)
To run these automation scripts on your machine, follow these direct, step-by-step instructions:

1. Download this repository using the green **Code** button at the top right, then select **Download ZIP**.
2. Unzip the folder on your machine.
3. Open your terminal or command prompt application.
4. Navigate to the folder using: `cd custom-developer-toolbox`
5. Run the target automation utility by typing: `python automation_utils/file_cleaner.py`
